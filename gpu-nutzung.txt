Bitte die Nutzung der Grafikkarte rechtzeitig planen und eintragen. Verwenden Sie eine weitere Zeile, wenn Sie bereits wissen, dass Sie eine GPU benötigen, sobald diese verfügbar wird.
Nach der Nutzung wieder entfernen.

Die aktuelle Auslastung kann mit "nvidia-smi" in der Konsole oder über Services->GPU-Auslastung (https://ki-server.oth-aw.de/services/GPU-Auslastung/) überprüft werden.
Wenn mehrere GPUs benötigt werden, pro GPU angeben.
Wenn ihr eine GPU benötigt, aber keine frei ist, schreibt Personen an.

If several GPUs are required, specify per GPU.
If you need a GPU but none is free, write to people.
